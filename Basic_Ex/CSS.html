CSS stands for Cascading Style Sheets.Wits CSS, you can control the color, font, the size of text, the spacing between elements,
how elements are positioned and laid out, what background images or background colors are to be used,
different displays for different devices and screen sizes, and much more!

Inline CSS

An inline CSS uses the style attribute of an HTML element.

Internal CSS

An internal CSS is used to define a style of single HTML page.
It's defined in the head section of and HTML page, within style element.

External CSS

An external style sheet is used to define the style for many HTML pages.
To use it, add a link to it in the head section of each HTML page.
